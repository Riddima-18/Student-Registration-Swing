# Student Registration Form (Java Swing)

A simple GUI-based Student Registration form built using Java Swing.

## Features
- Input fields for Name, Roll Number, Branch
- Gender selection (radio buttons)
- Terms & Conditions checkbox
- Submit and Reset functionality

## Technologies
- Java
- Swing (javax.swing)

## How to Run
1. Compile: `javac StudentRegistration.java`
2. Run: `java StudentRegistration`
